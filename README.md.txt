# Tableau de Bord CDLJ

Application de gestion des compositions pour la Communauté Diocésaine des Lecteurs Juniors de l'Archidiocèse de Cotonou.

## Fonctionnalités

- Gestion de deux activités distinctes
- Génération automatique des matricules
- Correction et analyse des notes
- Tableau de bord interactif
- Export des résultats

## Déploiement

Déployé sur Streamlit Cloud - Accès public
